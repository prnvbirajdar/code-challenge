## File Structure

- `lambda.js`: Contains the refactored AWS Lambda function.
- `server.js`: Server-side refactored code using Express.js
- `index.html`: The HTML document with an output container.
- `script.js`: Frontend script to fetch and display the data.

## Getting Started

Follow these steps to test and run the project locally:

### **1. Clone the Repository**
```
git clone https://github.com/prnvbirajdar/code-challenge.git
cd code-challenge
```

### **2. Install Dependencies**

Install dependencies in the repository.

```
npm install
```

### **3. Run Server**

Run the server locally.

Visit the application at http://localhost:3000

View the JSON at http://localhost:3000/get-array

```
npm run dev
```

### **4. Run Tests**

Run tests locally.

```
npm test
```
